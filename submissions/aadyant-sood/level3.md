## Description
This project is a UI design for a personal digital twin system that tracks health metrics like sleep, energy, and stress and provides AI-powered insights.

## LPI Tools Used (Conceptually)
- smile_overview
- query_knowledge
- analyze_patterns

## How It Works
The system takes user inputs such as sleep, energy, and stress and processes them to generate insights based on patterns.

## Example Interaction
Input: Sleep = 5 hours, Energy = Low  
Output: “Your energy dips after 2 PM, consider taking a short break.”

## Setup Instructions
1. Open the Figma design using the provided link  
2. View the dashboard layout  
3. Understand how insights are generated based on inputs  

## Design Approach

Although this is a UI prototype, it is designed as a system that mimics agent behavior.

The system:
- accepts inputs (sleep, energy, stress)
- processes them using logic
- generates insights

This approach aligns with how digital twin systems function.
